What do these limits mean?
AI Agents:
The number of different AI assistants you can create. Each agent can be specialized for different tasks like customer support, content writing, or data analysis.
API Calls:
Each time someone interacts with your AI agents (via API, web interface, or integrations), it counts as one API call. Resets monthly.
Custom Domains:
Websites your AI agents can access to extract information, learn from content, or monitor for updates. Each unique domain counts toward your limit.
